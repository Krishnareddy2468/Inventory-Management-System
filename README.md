# Inventory Management System

A full-stack **backend-driven inventory management system** designed to
replace manual inventory tracking with a structured, scalable solution.

---

## Problem
Manual inventory tracking leads to data inconsistency, poor visibility,
and inefficient stock management.

---

## Solution
Built a backend-centric system that enables structured inventory
management with reliable data storage and easy querying.

---

## Key Features
- CRUD operations for inventory items
- Search and pagination for large datasets
- CSV export for reporting and backups
- Role-friendly backend APIs

---

## Tech Stack
- Java
- Spring Boot
- MySQL
- REST APIs

---

## What I Learned
- Designing scalable CRUD-based backend systems
- Database integration and query handling
- API-driven application architecture
